¿Qué es Javascript y en qué se asemeja a Java?

Javascript y Java no se asemejan en nada a excepcion de que comparten una palabra al principio de su nombre,
JavaScript tradicionalmente ha sido un lenguaje interpretado, y Java es compilado.
Para entendernos, y dicho muy básicamente, los programas JavaScript son archivos de texto que pueden leer
tanto los ordenadores como las personas y que se interpretan a medida que se leen para luego ejecutarlos,
mientras que los de Java se compilan a un archivo especial optimizado para que lo lea un ordenador y lo ejecute.
